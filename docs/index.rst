## プロジェクト概要

これはSphinxとRead the Docsを使って作成されたドキュメントのサンプルです。
GitHubでソースコードを管理し、Read the Docsで自動公開しています。

**特徴:**

*   ウェブ上で完結する編集フロー
*   自動ビルド・デプロイ
*   分かりやすいドキュメント作成

より詳細な情報は、各セクションを参照してください。
